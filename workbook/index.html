Critiques on Stitch from user perspective:

I think Stitch is just a product at the middle of AI development timeline. It does not “perfectly” generates what I expected, even sometimes goes way off from what I ask it to do.

I would position Stitch currently as a tool for quick generative tool, just like ChatGPT. The difference between Stitch and ChatGPT is that Stitch has a lot more flexibility for designers to edit / transport assets and information. For example, the 12 hour challenge, that was one of experiment with Stitch. I’d like to see how Stitch react to a purely design task (in the situation, meaning not a webpage design, but infographic). The outcome was not bad, but it still has stick to the frame of “web” interface design, where on the design has buttons and footer etc. In the later iteration, it still cannot explicitly came up with a “pure information” art board that only contain the text and some graphics, which I cannot use Stitch entirely in this circumstance. HOWEVER! Stitch has the ability to let me copy it to Figma (which is my best friend recently), so I can edit them in external tool. This feature is very powerful compare to other AI tool that I used, and makes the design process a bit more efficient.

"Rose Land" is one of the movie I enjoyed watching. I did not know it has its own website until this time for gathering case study for interactive websites.

One unique characteristic of this website, is it minimise the way that user "interact" with webpage. Yes, it does has some interactive elements, but they are not clicking / hovering / dragging, just simply "scroll". One thing that I forgot when I do design, is that people sometimes scroll more over clicking and hovering elements. In my many works before, I often has interaction ready for user to click, but not when scroll.

"Rose Land" only display the most precise information that they want to display, and it only allows user to "scroll" over information, at some point, user can receive more information over having more freedom in the website (a bit contradicting with the point I made about "Marga Navarro" website). The percise information are also reinforced by the scrolling text effect which it highlight (added with colour) when user scroll through.

This kind of strategy ties user's attention and control the pace for the audience, it maximise the coverage of information pass through users.
